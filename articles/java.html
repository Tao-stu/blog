🔧 环境变量配置

1. 新建JAVA_HOME环境变量

  • 变量名: JAVA_HOME
  • 变量值: Java安装路径（例如：C:\Program Files\Java\jdk-17.0.2）

2. 配置Path环境变量

在系统环境变量Path中添加以下路径：

%JAVA_HOME%\bin

💡 配置步骤详解

Windows系统配置步骤：

  1. 打开环境变量设置
    • 右键点击"此电脑" → "属性" → "高级系统设置"
    • 点击"环境变量"按钮
  2. 配置JAVA_HOME
    • 在"系统变量"区域点击"新建"
    • 变量名：JAVA_HOME
    • 变量值：你的Java安装路径
  3. 配置Path变量
    • 找到并选中"Path"变量，点击"编辑"
    • 点击"新建"，添加：%JAVA_HOME%\bin
  4. 验证配置
    • 打开命令提示符(CMD)
    • 输入 java -version 查看Java版本
    • 输入 javac -version 查看编译器版本

✅ 验证方法

在命令提示符中运行以下命令验证配置是否成功：

# 检查Java版本
java -version

# 检查Java编译器版本
javac -version

# 检查JAVA_HOME环境变量
echo %JAVA_HOME%
