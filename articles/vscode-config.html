设置技巧

主题推荐：Dracula Theme 或 One Dark Pro
自动保存：设置 → 文件 → Auto Save → afterDelay
格式化代码：设置 → 文本编辑器 → 格式化 → 勾选 Format on Paste 和 Format on Save
字体大小：设置 → 文本编辑器 → Font Size
关闭鼠标悬停提示：设置 → 搜索 editor.hover → 取消勾选 Enable

Chinese (Simplified) Language Pack：中文语言包

Live Server：实时预览网页效果
Live Preview：在 VSCode 内预览网页
Auto Rename Tag：修改开始标签时自动同步结束标签
